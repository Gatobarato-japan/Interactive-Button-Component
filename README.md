# Reactive Chaos Button

Componente de interfaz interactivo que experimenta con comportamientos dinámicos y no convencionales en un botón HTML.

## Concepto
Este proyecto explora cómo un botón puede responder de forma creativa a la interacción del usuario,
generando efectos visuales y movimientos inesperados controlados por eventos de JavaScript.

## Features
- Reacción dinámica al hover
- Animaciones no lineales
- Respuestas visuales en tiempo real
- Experiencia interactiva experimental

## Tecnologías
- HTML5
- CSS3 (animaciones y transiciones)
- JavaScript (event handling)

## Objetivo
Explorar interacciones UI más expresivas y creativas que las interfaces tradicionales.
